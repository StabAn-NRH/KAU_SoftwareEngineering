# 지도 위에 쓰다, OnMap 

지도 서비스에는 수많은 정보와 장소가 표시되지만, 사용자는 지도에 있는 모든 정보와 장소에 관심 있어 하지는 않습니다. 

또한 장소에 관한 메모나 의견 등 사용자만이 만들어 낼 수 있는 다양한 종류의 정보들이 있지만 이를 제대로 표시해 주는 지도 서비스가 거의 없는 실정입니다.

**OnMap** 프로젝트는 현재 다양한 지도 서비스에서 제공하는 관심 있는 장소 표시 기능에서 한 걸음 더 나아가, 지도에 사용자가 관심 있는 다양한 종류의 정보를 표시하는 사용자화 지도 서비스를 제공하고 이를 통해 지도 서비스의 개선 방향을 제시하고자 합니다.

**OnMap** 프로젝트는 다음과 같은 목적을 가지고 있습니다.
* 더 많은 장소 정보의 제공
* 위치 정보의 가시성 향상
* 지도의 사용자화

<br>

**\* 이 프로젝트는 소프트웨어공학 과목의 형상 관리 과제를 위한 가상의 프로젝트입니다.**
